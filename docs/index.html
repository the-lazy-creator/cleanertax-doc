Essência do Produto (MVP)

Todos que possuem investimentos em ações ou fundos imobiliários precisam incluir algumas informações na declaração anual do imposto de renda.

Dentre essas informações inclui-se o ganho com dividendos para cada ação, a quantidade de ações, o preço médio de compra/custo médio de aquisição (PS: nesse momento não consideraremos o preço de compra de trade) para cada uma dessas ações, as operações de compra e venda, o lucro ou o prejuízo das operações (o mesmo se aplica para os fundos imobiliários).

O investidor possui todas essas informações porém pode ser dificil juntar todas essas informações quando se tem muitos ativos. Desse modo, a ideia é criar um software que receba de input todas as operações realizadas no decorrer do ano e que forneça o modelo pronto para declaração.
